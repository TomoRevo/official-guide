AIは優秀なパートナーだけど、白紙の状態から「何か聞いて」と言われても、手が止まる。これは当然のことで、あなたの能力の問題ではない。

料理でいえば、レシピがあれば作れるのに、「何か作って」と言われると困るのと同じ。まずはレシピ（テンプレート）を手に入れて、そのまま使ってみる。それが一番早い。

ここでは、事業を運営しているあなたが今日から使える5つのテンプレートを用意しました。やることはシンプル。【 】の部分を自分の事業に合わせて書き換えて、AIに送るだけ。

スライド資料を作るときにも同じ原則が使えます。「10枚のスライドを一気に作って」と頼むと処理が多すぎて質が落ちる。「まず構成を作って」→「1枚目を作って」と分けて進めるのがコツ。テンプレートも同じで、1つずつ試して感覚をつかむのが一番確実です。

TEMPLATE 01

クライアントへの提案メール

伝えたいことはあるのに文章にまとめる時間がない。要点だけ箇条書きにしてAIに渡せば、丁寧なメールが数秒で仕上がる。

プロンプト（AIへの指示文）
以下の情報をもとに、クライアントへの提案メールを作成してください。 ■ 送信者の業種: 【例: 税理士事務所】 ■ サービス名: 【例: 山田税理士事務所】 ■ 送り先: 【例: 飲食店を3店舗経営する個人事業主】 ■ 提案内容: 【例: 確定申告だけでなく、月次の経営数字の見える化サポート】 ■ 相手のメリット: 【例: 毎月の利益がリアルタイムでわかる、節税ポイントを早めに把握できる】 ■ トーン: 【丁寧 / カジュアル / フォーマル】 ■ メールの長さ: 【200字程度 / 400字程度】 件名も含めて作成してください。 押し売りにならず、相手の課題解決を起点にした構成にしてください。
AIの回答
件名: 毎月の経営数字を「見える化」するサポートのご提案

〇〇様

はじめまして。山田税理士事務所の山田と申します。

飲食店を複数店舗経営されていると、日々の売上管理や経費の把握にお忙しいかと存じます。確定申告の時期にまとめて整理するのではなく、月次で経営数字を見える化することで、利益の変動や節税のタイミングを早めに把握できるようになります。

当事務所では、店舗ビジネスに特化した月次レポートの仕組みをご提供しており、「数字を見る習慣ができた」「仕入れのムダに早く気づけた」というお声をいただいております。

まずは15分ほど、現在の管理方法についてお話を伺えればと思います。ご都合のよい日時をお知らせいただけますでしょうか。

何卒よろしくお願いいたします。

ポイント: 「送り先」と「相手のメリット」を変えるだけで、まったく違うメールが出てくる。誰向けのメールかを伝えるだけで、言葉遣いも構成も変わる。

TEMPLATE 02

SNS投稿文の生成

SNSの投稿、毎回ゼロから考えるのは地味に時間がかかる。ターゲットとトーンを指定するだけで、そのまま使える投稿文が手に入る。

プロンプト（AIへの指示文）
以下の条件でSNS投稿文を作成してください。 ■ 投稿するSNS: 【Instagram / X（Twitter）/ LINE公式】 ■ 業種: 【例: ネイルサロン】 ■ ターゲット: 【例: 30代の働く女性、仕事帰りに自分へのご褒美を求めている】 ■ 投稿の目的: 【認知拡大 / 来店促進 / 信頼構築】 ■ 伝えたい内容: 【例: 新しい春デザインが入荷した】 ■ トーン: 【親しみやすい / 上品 / 元気】 ■ 文字数: 【150字以内 / 300字以内】 ■ ハッシュタグ: 【つける（5個程度） / つけない】 「売り込み感」が出ないように、共感や日常の一場面から入る構成にしてください。

仕事終わり、電車のなかで自分の指先を見て
「そろそろ変えたいな」って思う瞬間、ありませんか。

春の新作デザイン、入りました。
今季は淡いラベンダーとシアーなベージュが人気です。

頑張った自分に、指先から小さなご褒美を。
プロフィールのリンクからご予約いただけます。

#ネイルデザイン #春ネイル #ご褒美ネイル #大阪ネイルサロン #新作デザイン

ポイント: 「投稿の目的」を変えると構成がガラッと変わる。認知拡大なら共感重視、来店促進ならCTA（行動喚起）重視の文面になる。

TEMPLATE 03

リサーチ業務をAIに任せる

市場調査、競合の特徴整理、新しいサービスを始める前の情報収集。リサーチは大事だけど、自分でやると半日がすぐ消える。情報を集める作業は人間がやるべきではない。あなたのリソースは「意思決定」に使うべき。集めて整理するところまではAIに任せよう。

プロンプト（AIへの指示文）
以下のテーマについて、事業判断に必要な情報を整理してください。 ■ 調べたいテーマ: 【例: 自分のエステサロンの近隣にある競合サロンの特徴】 ■ 自分の事業の概要: 【例: 個人経営のフェイシャルエステサロン、開業2年目、完全予約制】 ■ 知りたいこと: - 【例: 近隣エリアの競合はどんなメニュー構成・価格帯が多いか】 - 【例: 自分のサロンの強みと弱みは何か】 - 【例: 差別化できるポイントはどこか】 ■ 最終的にやりたいこと: 【例: 来月の新メニュー追加の判断材料にしたい】 以下の形式で出力してください: 1. 調査結果の要点（箇条書き） 2. 比較表（表形式） 3. あなたの事業への提案（3つ）
AIの回答

調査結果の要点:

  • フェイシャルエステの一般的な価格帯は初回5,000〜8,000円、通常8,000〜15,000円
  • 大手チェーンは低価格×回数券で集客、個人サロンは「専門特化」で差別化する傾向
  • SNSでの口コミ・ビフォーアフター投稿が集客の主戦場になっている
観点自分のサロン大手チェーン近隣の個人サロン
価格帯中価格（10,000円前後）低価格（5,000円〜）中〜高価格
強み完全予約制・マンツーマン駅近・知名度専門技術・こだわり
集客紹介中心広告・クーポンサイトInstagram・口コミ

あなたの事業への提案:

  • 「完全予約制×マンツーマン」は大手にない強み。価格競争ではなく体験の質で勝負する
  • 新メニューは既存客の悩みベースで設計すると、リピート率が上がりやすい
  • ビフォーアフター投稿を月2回でもInstagramに出すと、紹介以外の流入が増える可能性
ポイント: リサーチの目的は「情報を集めること」ではなく「意思決定すること」。AIに整理させた結果を見て、判断するのはあなた。集める作業に半日使うのではなく、判断に集中できる状態を作ろう。競合だけでなく、新サービスの市場調査、イベント企画の情報収集など、リサーチ全般に同じテンプレートが使える。

TEMPLATE 04

サービス紹介文・キャッチコピー

ホームページ、チラシ、SNSのプロフィール。サービスの紹介文を書く場面は多いのに、自分のことほど書きにくい。

プロンプト（AIへの指示文）
以下の情報をもとに、サービス紹介文とキャッチコピーを作成してください。 ■ サービス名: 【例: ITコーチング by TomoRevo】 ■ 何をするサービスか: 【例: IT苦手な事業主にAI活用を伴走型でサポートする】 ■ ターゲット: 【例: 1人で何役もこなしている個人事業主・小規模経営者】 ■ ターゲットの悩み: 【例: AIに興味はあるけど何から始めればいいかわからない、時間がない】 ■ このサービスを受けた後の変化: 【例: 日常業務の一部をAIに任せられるようになり、本来やりたい仕事に集中できる】 ■ 使用場所: 【ホームページ / チラシ / SNSプロフィール】 ■ トーン: 【信頼感重視 / 親しみやすさ重視 / 挑戦的】 以下を出力してください: 1. キャッチコピー（3案）— 各15字以内 2. サービス紹介文（200字以内） 3. サービス紹介文（80字以内・SNS用）
AIの回答
キャッチコピー案:
  • 1. AIを、あなたの右腕に。
  • 2. 苦手なITは、任せる時代へ。
  • 3. 経営に集中できる毎日を。
サービス紹介文（200字）:

「AIを使いたいけど、何から手をつければいいかわからない」そんな事業主の方に、あなたの事業に合ったAI活用を一緒に設計し、導入まで伴走するサービスです。汎用的な講座ではなく、あなたの日常業務を起点に、実際に使える形でAIを組み込みます。メール作成、リサーチ、資料づくり——毎日の「これ、時間かかるな」が、数分に変わる体験を。

SNS用（80字）:

IT苦手な事業主に、AI活用を伴走サポート。あなたの事業に合わせて、日常業務にAIを組み込みます。

ポイント: 「使用場所」を変えれば、同じサービスでもホームページ向けの丁寧な文章と、SNS向けの短い文章の両方が手に入る。

TEMPLATE 05

顧客対応の定型文

問い合わせへの返信、お礼のメッセージ、お断りの連絡。毎回ゼロから書くと時間もかかるし、気も使う。テンプレートを用意しておけば、質を保ちながら対応スピードが上がる。

プロンプト（AIへの指示文）
以下の条件で、顧客対応の定型文を作成してください。 ■ 業種: 【例: ヨガスタジオ】 ■ 場面: 【問い合わせ返信 / 来店後のお礼 / 予約変更の案内 / お断り】 ■ 連絡手段: 【メール / LINE / DM】 ■ 相手: 【新規の問い合わせ客 / 既存の顧客 / 紹介経由の方】 ■ 伝えたい内容: 【例: 体験レッスンの日程案内と持ち物の連絡】 ■ トーン: 【温かみのある丁寧語 / ビジネスライク / フランク】 ■ 補足: 【例: 動きやすい服装ならOK、マットは無料レンタルと伝えたい】 短すぎず長すぎず、相手が安心できる文面にしてください。

ご都合のよい日時をお知らせいただければ、ご予約を確定いたします。

お会いできることを楽しみにしております。

ポイント: 「場面」を変えるだけで、お礼・案内・お断りなど様々な文面が出てくる。特にお断りの文面は気を使いすぎて時間がかかりがち。AIに土台を作ってもらい、自分の言葉で仕上げる。それだけで精神的な負担も作業時間も大幅に減る。
